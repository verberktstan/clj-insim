parse
(parse coll)
Returns a set of items from `coll` for a given integer `i`.
 This is useful to parse bit-flags.
 ```clojure
(parse [:a :b] 2) => #{:b}`
(parse [:a :b] 3) => #{:a :b}`
```

unparse
(unparse coll)
Returns integer representation of flags.
```clojure
(unparse [:a :b :c] #{:b}) => 2
(unparse [:a :b :c] #{:b}) => 2
```
